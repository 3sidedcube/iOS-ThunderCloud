TSCStormObject.m

Overview

A subclass of NSObject which is the base class of all storm objects. This class is used primarily for overriding standard storm behaviour.

+ overideClass:with:

This method is used to override the class instantiated for any Storm base class.

+ (void)overideClass:(Class)originalClass with:(Class)newClass

Parameters

newClass

The class to instantiate in replacement of originalClass
originClass

The class to override.

Discussion

This method allows customising storm standard behaviours. Overriding a class here means that whenever Storm instantiates a class of this type it will use your replacement class. The overriding class should almost always subclass from the overidden class to ensure behaviour of your app isn’t effected.
